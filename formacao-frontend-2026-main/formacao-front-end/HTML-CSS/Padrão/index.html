Cores

Temos 4 tipos de cores do css

De Forma escrita exemplo color:red

De forma hexadecimal exemplo color #ff0000

De forma hexadecimal exemplo color #0000

De forma RGB exemplo color: rgb(255,0,0)

De Forma RGBA exemplo color:rgba (255,0,0,0.5)

De Forma HSL color: hsl(0,100%,50%)

H = Hue (matriz de Tonalidade)

S = Saturation ( saturação)

L = Lightness ( Luminosidade)

De Forma HSLA color: hsla(0,100%,50%,50%)
